## Meaning

프론트, 백엔드 최신 기술 스택이 미숙한 둘이서 실력 향상을 위한 가장 기본적인 CRUD인 블로그를 구현해보는 프로젝트

<br/>

## PurPose
- axios를 이용하여 REST API 호출에 대한 이해 및 CRUD 구현
- custom hook 구현
- react-router를 이용하여 웹 페이지 접근 제어 이해 및 구현
- react-query를 활용하여 서버상태 동기화
- redux-toolkit을 이용한 전역 상태 관리
- redux-persist를 활용하여 로그인 상태 유지 기능 구현
- toast-ui를 사용한 마크다운 에디터 구현
- tailwindcss를 활용한 반응형 웹 디자인
- 코드 품질 향상을 위한 eslint, prettier 적용
